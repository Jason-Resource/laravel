- 1.创建包含自定义函数的php文件
```
#路径 : Laravel根目录/app/helpers.php
<?php
// 检查是否定义，避免冲突
if( ! function_exists('my_test') ){
    function my_test(){
        return 'hello world';
    }
}

?>
```

- 2.修改composer.json的autoload配置（自动引入文件）
```
#路径 : Laravel根目录/composer.json
#找到composer.json的autoload项，在files中加入欲引入文件
    "autoload": {
        ...
        ...
        "files" : [
            "app/helpers.php" // 引入helpers.php文件
        ]
    },
```
